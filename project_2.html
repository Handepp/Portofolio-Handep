Sistem Monitoring dan Kendali Berbasis GUI

Sebuah sistem yang dirancang untuk memantau laju air, suhu, dan humiditas serta mengendalikan motor DC melalui antarmuka GUI dengan menggunakan protokol komunikasi MQTT

Proyek ini saya kerjakan ketika sedang magang di PT. Syergie Indo Prima sebagai bagian dari pemenuhan nilai mata kuliah kerja praktek industri (KPI). Selama magang, saya dibimbing untuk mengembangkan sebuah sistem monitoring dan kendali yang dapat diimplementasikan ke dalam GUI.

Pada GUI ini dilakukan monitoring terhadap laju air, suhu, dan humiditas, serta mengendalikan motor DC. Sistem ini dibangun menggunakan beberapa komponen utama, yaitu:

  • Bahasa Pemrograman Python: Digunakan untuk menerima data dari Arduino IDE yang dikirimkan melalui MQTT, lalu menghubungkannya dengan GUI (Back-end)
  • Bahasa Antarmuka QML: Digunakan untuk membangun tampilan Graphical User Interface (GUI) yang interaktif dan mudah digunakan (Front-end).
  • ESP32 sebagai Mikrokontroler: Berperan sebagai otak dari sistem yang mengumpulkan data dari sensor dan mengirim serta menerima data melalui MQTT. Logika pada ESP32 dibuat di Arduino IDE.
  • Sensor Laju Air: Digunakan untuk mengukur laju aliran air dalam sistem.
  • Sensor DHT11: Sensor ini digunakan untuk mengukur suhu dan kelembaban udara.
  • Motor Driver L298N dan Motor DC: Digunakan untuk mengendalikan pergerakan motor DC berdasarkan perintah yang diterima dari GUI.
  • Protokol Komunikasi MQTT: Digunakan untuk komunikasi antara perangkat keras (ESP32 dan sensor) dengan GUI, memungkinkan pengiriman data secara real-time.

Selama proses pengembangan, saya menggunakan ESP32 untuk mengumpulkan data dari sensor laju air dan sensor DHT11. Data ini kemudian dikirim ke Python melalui protokol MQTT, kemudian dikirim lagi ke GUI yang dibangun dengan QML. GUI ini memungkikan menampilkan data perangkat keras yang diterima secara real-time. Selain itu, GUI ini memungkinkan pengguna untuk mengendalikan kecepatan dan arah motor DC. Perintah dikirim melalui GUI (QML) ke Python, kemudian diteruskan ke ESP32 melalui protokol MQTT.

Berikut merupakan Graphical User Interface (GUI) yang dikembangkan untuk sistem monitoring dan kontrol yang saya kerjakan dalam proyek di PT. Syergie Indo Prima. GUI ini dirancang untuk memantau dan mengendalikan berbagai parameter pada sistem berbasis sensor dan motor.

Water Flow Sensor Monitoring

GUI ini menampilkan data real-time dari sensor aliran air. Parameter yang dipantau meliputi laju aliran air (Flow Rate) dan total volume air yang telah mengalir (Total Volume). Selain itu, terdapat juga grafik yang menunjukkan tren aliran air seiring waktu. Akan tetapi, pada hasil GUI ini masih belum dapat menampilkan grafik yang benar terhadap waktu yang dilaluinya. Data yang ditampilkan pada GUI ini juga disimpan pada file excel menggunakan bahasa python.

Temperature and Humidity Sensor Monitoring

GUI ini digunakan untuk memantau kondisi suhu dan kelembaban dari sensor DHT11. Data yang ditampilkan termasuk suhu dalam derajat Celsius dan kelembaban dalam persen. Grafik di bagian bawah menggambarkan perubahan suhu dan kelembaban dari waktu ke waktu. Sama seperti halaman untuk water flow sensor, pada hasil GUI ini masih belum dapat menampilkan grafik yang benar terhadap waktu yang dilaluinya. Data yang ditampilkan pada GUI ini juga disimpan pada file excel menggunakan bahasa python.

DC Motor Control

GUI ini berfungsi untuk mengontrol kecepatan motor DC serta memonitor parameter terkait. Terdapat dua motor DC yang dapat diatur kecepatannya melalui slider, dengan mode arah putaran (CW/CCW) juga bisa diatur dari interface ini. Nilai PWM (Pulse Width Modulation) menunjukkan kecepatan motor dalam satuan yang relevan.
